==========
User Guide
==========

Welcome to the Construbot User Guide! This section is designed for construction company staff who use Construbot
to manage projects, track estimates, and handle contract administration.

.. note::
   **New to Construbot?** Start with :doc:`getting-started` to learn the basics.

What is Construbot?
===================

Construbot is an operational management solution for construction companies. It helps you:

- **Manage Contracts** - Track construction projects from start to finish
- **Monitor Progress** - Create and manage progress estimates (estimaciones)
- **Financial Tracking** - Calculate payments, retentions, and advance amortization
- **Multi-Company Support** - Work across multiple business entities
- **Document Generation** - Generate professional PDF estimates and reports

Who Should Use This Guide?
===========================

This guide is for:

- Project Managers
- Site Supervisors
- Contract Administrators
- Accounting Staff
- Directors and Coordinators
