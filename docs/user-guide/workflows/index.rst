=========
Workflows
=========

Step-by-step guides for common tasks in Construbot.

Overview
========

This section provides detailed workflows for performing common operations. Each workflow includes prerequisites, step-by-step instructions, and troubleshooting tips.

Common Workflows
================

Creating a New Project
----------------------

**Complete workflow from client setup to first estimate:**

1. **Create client counterparty** (Proyectos → Contrapartes)
2. **Add project site** (Proyectos → Sitios)
3. **Create contract** with details and retentions
4. **Build concept catalog** (manual or Excel import)
5. **Add document recipients** (contacts)
6. **Create first estimate** for progress billing

See :doc:`/user-guide/concepts/projects-contracts` for detailed information.

Managing Estimates
------------------

**Monthly progress billing workflow:**

1. Navigate to contract
2. Click "Nueva Estimación"
3. Set period dates
4. Add line items with quantities completed
5. Review calculations and retentions
6. Generate PDF document
7. Submit for approval

See :doc:`/user-guide/concepts/estimates` for details.

Setting Up Subcontracts
-----------------------

**Hierarchical contract structure:**

1. Create parent contract (main client contract)
2. Create subcontractor counterparty
3. Create subcontract linked to parent
4. Manage separate estimates for each level

See :doc:`/user-guide/concepts/projects-contracts` (Hierarchical Contracts section).

Importing Catalogs
------------------

**Bulk import via Excel:**

1. Prepare Excel with columns: Code, Description, Unit, Quantity, Unit Price
2. In contract, click "Import from Excel"
3. Map columns and verify preview
4. Confirm import

See :doc:`/user-guide/concepts/concepts-line-items` for catalog management.

Tips for Efficiency
===================

- Use template contracts for similar projects
- Import catalogs via Excel for large projects
- Keep counterparty information current
- Review estimates before generating PDFs
